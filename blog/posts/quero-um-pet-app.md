Unindo tecnologia e uma causa nobre, desenvolvi o **Quero Um Pet**, um aplicativo mobile criado para otimizar a rotina de ONGs de adoção de animais.  
O objetivo foi construir uma ferramenta centralizada, intuitiva e funcional para facilitar tarefas essenciais dessas instituições.

Construído com **React Native + Expo**, o app oferece um sistema completo de gerenciamento, permitindo controle de pets, adoções e finanças — tudo em um único lugar.

## Principais funcionalidades implementadas

### **Autenticação segura**
- Login com e-mail e senha.  
- Integração com Google Auth via Firebase.  
- Persistência de sessão utilizando AsyncStorage para manter o usuário conectado.

### **Gestão completa de pets**
- CRUD completo para cadastro e edição de animais.  
- Suporte a fotos, informações detalhadas e status de adoção.

### **Controle de adoções**
- Registro e consulta de adoções realizadas.  
- Cada adoção é vinculada ao perfil do animal e aos dados do adotante.

### **Gestão financeira integrada**
- Módulo para registrar entradas e saídas.  
- Cálculo automático de saldo para um controle mais transparente.

### **Navegação fluida**
- Interface organizada com Tab Navigator.  
- Ícones, layout customizado e foco total na experiência do usuário.

## Um projeto com aprendizado real

Este aplicativo foi apresentado na disciplina de **Desenvolvimento para Dispositivos Móveis**, no curso de Ciência da Computação da UNIPAC.  
O projeto permitiu aprofundar conhecimentos em desenvolvimento mobile — desde autenticação e persistência de dados até a criação de interfaces funcionais e agradáveis.

Tecnologia com propósito sempre faz diferença. E esse projeto foi exatamente sobre isso. 🐾
